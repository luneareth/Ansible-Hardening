CRONCAT APPLICATION NEAR BLOCKCHAIN
=========

Autodeploy Croncat APP with ansible

Requirements
------------

For Server

```
vCPU 1
RAM 512
Disk 30GB
```

1. Nodejs
2. Curl

Role Variables
--------------

```
user_app_croncat: croncat
```
as user app.
